这篇论文提出了 Transformer，用自注意力机制替代循环和卷积结构，成为后续大语言模型的重要基础。

## 背景与问题

传统序列建模方法常依赖 RNN 或 CNN，长距离依赖建模和并行训练都存在限制。

## 核心方法

- Multi-Head Attention
- Positional Encoding
- Encoder-Decoder 架构
- Scaled Dot-Product Attention
